# Support

Use GitHub Issues for bugs, documentation problems, compatibility reports, and feature requests.

When reporting a bug, include:

- `robyn-mcp` version
- Python version
- Robyn version, if installed
- MCP client and version
- Transport used
- Minimal reproduction
- Output from `robyn-mcp runtime --json`
- Output from `robyn-mcp doctor <endpoint> --json`, when a live endpoint is involved

Do not include tokens, credentials, cookies, private headers, or production customer data.
